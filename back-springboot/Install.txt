Sans IntelliJ Ultimate : Télécharger zip sur https://start.spring.io/

Avec IntelliJ Ultimate : Créer un nouveau projet "Java Spring".
- Type : Maven

Ajouter les dépendances :
- Spring Web
- Spring Boot DevTools
- Validation
- Spring Data JPA
- MySQL Driver
- PostgreSQL Driver

- Lombok
- BCrypt


----- Vérifier ce qu'on a dans pom.xml : -----

----- Tester la connexion à la BDD (facultatif) : -----

- Démarrer PostgreSQL ou MySQL.
- Dans IntelliJ, cliquer sur l'icône "Database" à droite. Puis ajouter PostgreSQL ou MySQ.
- Remplir User et Password.
    Dans PostgreSQL avec PgAdmin :
      - User : postgres
      - Password : mot de passe qu'on a défini
    Dans MySQL avec PhpMyAdmin :
      - User : root
      - Password : Laisser vide
- Cliquer sur "Test Connection", puis "Download Driver Files".
- Mettre de nouveau le mot de passe, puis Ok.
- Créer une BDD manuellement dans PgAdmin ou PhpMyAdmin.
- Ouvrir le fichier "application.properties" et mettre :


----- Fichier application.properties : -----

spring.application.name=back-springboot

# Creer d'abord la BDD manuellement

#spring.datasource.url=jdbc:postgresql://localhost:5432/recette
#spring.datasource.username=postgres
#spring.datasource.password=1234
#spring.jpa.hibernate.ddl-auto=update

spring.datasource.url=jdbc:mysql://localhost:3306/recette
spring.datasource.username=root
spring.datasource.password=
spring.jpa.hibernate.ddl-auto=update
spring.jpa.database-platform=org.hibernate.dialect.MySQL8Dialect
